# insights
No-code / low-code administration system


Insights is a no-code / low-code administration system in development. Written in Php

Currently the basis for an administrationsystem is build with support for
- customers, persons & companies
- invoices
- project support
- API/gui for sending mails
- filebackup
- forms-system
- all database action are transactional
